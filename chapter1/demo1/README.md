## 基于 npm + webpack 的 ThreeJS 浏览器基础项目

## 安装

在项目描述文件`package.json`同级下目录依次执行以下命令：

```bash
npm install
npx webpack
```

## 目录介绍

```
├── dist                 //webpack 构建打包生成的源代码目录
├── src                  //基于 ES6 模块机制组织的源代码目录
├── node_modules         //npm 下载的第三方包目录
├── package.json         //npm 项目描述文件
├── index.css            //样式
└── index.html           //页面
```
